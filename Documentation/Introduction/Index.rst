What does it do?
----------------

The Avalex extension allows to display an automatically generated and updated „Data Privacy Statement” within a TYPO3 web site.
This requires an account at https://www.avalex.de/121.html . A Data Privacy Statement is required to comply with GDPR/DSGVO regulations.
NOTE: The Avalex Service is currently available in German language only. Therefore the generated Date Privacy Statemenent (Datenschutzerklärung) is only in German.
